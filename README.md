# 🎵 Music Player

A responsive and interactive Music Player built using **HTML**, **CSS**, and **JavaScript**. The application allows users to play, pause, switch songs, and control music playback with an attractive user interface.


## 🚀 Features


- ▶️ Play and Pause songs
- ⏭️ Next and Previous track navigation
- 🔊 Volume control
- 📊 Progress bar with seek functionality
- 🎶 Display song title and artist
- 📱 Fully responsive design
- 🎨 Modern and user-friendly interface


## 🛠️ Technologies Used

 HTML5
- CSS3
- JavaScript
